# kissflow-app-ui

Claude Code plugin that scaffolds and builds custom UIs for Kissflow apps.

- **Skill** `build-kissflow-app` — auto-invokes when you ask to build/create a Kissflow
  custom UI. Scaffolds via `@sooryakanth/create-kf-app`, installs, syncs the app's data
  models, reads the project docs, and builds the screens.
- **Command** `/kissflow-app-ui:new-kf-app <project-name> <App_ID>` — explicit scaffold + build.

See the marketplace [README](../../README.md) for install instructions.
